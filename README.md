# Zid - تطبيق طلبات الطعام

تطبيق متكامل لطلب الطعام من المطاعم والأكل البيتي في الشيخ زايد.

## 🚀 المميزات
- تصفح المطاعم والشيفات المنزلية
- مساعد ذكي (مُنجز) للاقتراحات
- سلة طلبات متكاملة
- تسجيل صوتي للطلبات
- لوحة تحكم للأدمن والتجار

## 🛠️ التقنيات المستخدمة
- React Native / Expo
- Appwrite (قاعدة البيانات)
- Gemini AI (المساعد الذكي)
- ImageKit (الصور والفيديو)

## 📦 التثبيت والتشغيل

npm install
expo start
